# Quizzo

## Prerequisties

* Python 3
* MySQL

## Setting up the Virtual Environment and Installing the Libraries

### Linux (Ubuntu)
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Windows
```
python3 -m venv venv
venv/Scripts/activate
pip install -r requirements.txt
```

## Run the code

```
python app.py
```
